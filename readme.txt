Character count, word count, and sentence count tool with shortcode [text_counter].

## Installation

1. Visit **Plugins > Add New**.
2. Search for **Text Counter Tool**.
3. Install and activate the Text Counter Tool plugin.

## Usage

Use the the shortcode [text_counter] antwhere in wordpress(pages, posts, custom posts) to count character, word, and sentence.

## Features

- Feature 1: Use shortcode.
- Feature 2: Count character, word, and sentence.
- Feature 3: No settings required.

## Configuration

No configuration required.
